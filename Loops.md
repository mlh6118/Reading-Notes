## Loops in JavaScript

Loops offer a quick and easy way to do something repeatedly.

Many different types of loops:
- for statement
  ```html
  for (initial expression; conditional expression; increment expression){ code block }
  ```
- do...while statement
  ```html
  do
    statement
  while (condition;)
  ```
- while statement
  ```html
  while (condition)
    statement
  ```
- labeled statement
  ```html
  label :
      statement
  ```
